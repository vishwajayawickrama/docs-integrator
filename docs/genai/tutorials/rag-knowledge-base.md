# Tutorial: RAG Knowledge Base

The end-to-end **RAG knowledge base** tutorial is now maintained as **[Building an HR Knowledge Base with RAG](/docs/genai/tutorials/building-hr-knowledge-base-rag)**. It walks through the same pipeline (Data Loader, Vector Knowledge Base, ingest, then retrieve, augment, generate) using the WSO2 Integrator visual designer and the built-in `ai` module. No third-party SDK code is required.

If you landed here from an older bookmark:

- For a hands-on RAG tutorial, follow **[Building an HR Knowledge Base with RAG](/docs/genai/tutorials/building-hr-knowledge-base-rag)**.

- For per-component options (vector stores, embedding providers, chunkers), see **[AI Connections and Stores](/docs/genai/develop/components/overview)**.
