# Coming from WSO2 MI

This guide helps existing WSO2 MI users get started with WSO2 Integrator. Whether you are building new integrations or considering migrating existing ones, this page maps familiar MI concepts to their WSO2 Integrator equivalents.

:::tip WSO2 MI Documentation
WSO2 MI continues to be fully supported. For MI-specific documentation, visit [mi.docs.wso2.com](https://mi.docs.wso2.com).
:::

## Concept Mapping

| In WSO2 MI | In WSO2 Integrator | Notes |
|---|---|---|
| Proxy Service | Service | HTTP/SOAP endpoints |
| Sequence | Integration logic | Inline in services |
| Mediator | Ballerina functions | Type-safe, compiled |
| Message Processor | Event Handler | Reactive processing |
| Scheduled Task | Automation | Cron-based triggers |
| Data Service | Service + DB Connector | Unified model |
| Registry | Config.toml | Simpler configuration |
