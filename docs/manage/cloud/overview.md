# WSO2 Cloud

After your integrations are deployed to WSO2 Cloud - Integration Platform, the cloud console gives you full control over how they run. You can inspect deployment and build status, manage the integration lifecycle, configure runtime values per environment, promote integrations through your release pipeline, and tune security and scaling, all without touching infrastructure.

## Management areas

| Area | Description |
|---|---|
| [Integrations](./integrations/viewing-deployed.md) | View deployment and build status, and manage the lifecycle of each integration — including schedules, redeployment, and stopping. |
| [Configuration](./configurations/overview.md) | Set runtime values and secrets, control endpoint visibility and security, configure build behavior, and tune autoscaling per environment. |
| [Environments](./environments/overview.md) | Promote integrations across environments with optional approval gates, and override configuration values at each stage. |
