Deploy & Operate

Everything from pushing your first build to production to monitoring it at scale. This section covers all deployment targets and integration types, including GenAI agents.

Where does Develop end?

If your code is still on your machine, you're in Develop. Once you're pushing it somewhere else, you're here.

Deploy​

Get your integration running in any environment:

TargetDescription
Run LocallyTest on your machine before deploying
VM-BasedStandalone and centralized (consolidated packages)
Docker & KubernetesContainerized deployments with orchestration
OpenShiftRed Hat OpenShift deployments
ServerlessAWS Lambda, Azure Functions
Push to CloudOne-click deploy from WSO2 Integrator IDE
Cloud ProvidersDeploy to AWS, Azure, or GCP
GraalVM NativeAhead-of-time compiled binaries for fast startup

Environment management: Environments | Managing Configurations | Scaling & High Availability

CI/CD​

Automate your build and deploy pipeline:

  • GitHub Actions — Workflows for build, test, and deploy
  • Jenkins — Pipeline configuration
  • GitLab CI — .gitlab-ci.yml setup
  • Azure DevOps — Azure Pipelines integration

Third-party integrations: Datadog | New Relic | Elastic Stack | OpenSearch

Secure​

Harden your integrations for production:

  • Runtime Security — JVM security, keystores, non-root execution
  • Authentication — OAuth 2.0, JWT, mTLS
  • API Security & Rate Limiting — Protect your endpoints
  • Secrets & Encryption — Keystores, environment variables, vault integration
  • Compliance — Compliance considerations
