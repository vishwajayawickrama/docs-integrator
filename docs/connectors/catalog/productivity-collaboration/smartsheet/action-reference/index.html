Actions

The ballerinax/smartsheet package exposes the following clients:

ClientPurpose
ClientManages all Smartsheet resources — sheets, rows, columns, folders, workspaces, users, reports, and more.

Client​

Manages all Smartsheet resources — sheets, rows, columns, folders, workspaces, users, reports, and more.

Configuration​

FieldTypeDefaultDescription
authhttp:BearerTokenConfig|OAuth2RefreshTokenGrantConfigRequiredBearer token config ({token: "..."}) or OAuth 2.0 refresh token config.
httpVersionHttpVersionHTTP_1_1HTTP protocol version.
timeoutdecimal60Request timeout in seconds.
retryConfigRetryConfig()Retry configuration for failed requests.
secureSocketClientSecureSocket()SSL/TLS configuration.
proxyProxyConfig()Proxy server configuration.

Initializing the client​

import ballerinax/smartsheet;

configurable string smartsheetToken = ?;

smartsheet:Client smartsheetClient = check new ({
    auth: {
        token: smartsheetToken
    }
});
